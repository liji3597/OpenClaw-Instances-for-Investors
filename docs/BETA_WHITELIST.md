# P5 Beta Whitelist Program / Beta 测试指南

## 1. Participation Limits / 参与限额
To ensure safety during the initial rollout, the following hard limits are enforced in the Gateway:
为确保初期启动安全，网关将强制执行以下硬性限制：
- **Max Users:** 10 (Invite-only)
- **Per Transaction Cap:** $100 USD (in SOL/USDC)
- **Daily Total Limit:** $500 USD per user

## 2. Application Process / 申请流程
1. User sends `/beta_apply` to the Telegram Bot.
2. User signs a message with their Solana wallet to prove ownership.
3. Admin reviews and updates `data/whitelist.json`.

## 3. Feedback Channels / 反馈渠道
- **Bug Report:** Use `/bug` command in the bot.
- **Emergency:** Contact the @OpenClaw_Support Telegram handle immediately.
